# HTML_CSS: 01-Refractor Code 

This repository contains an HTML and accompanying CSS file that has been refractored (i.e. it has been improved without changing what it does.) In areas were semantic tag changes are made in the HTML file, the CSS file is modified as well for ease of code reading. 

## Installation 

1. Install any web browser with HTML5 capabilities. 
2. Right click on the [index.html](index.html) and open in installed web browser 

## Changes made to source code: 

* HTML
  - Descriptive img alt tags. 
  - Company name in  title tag
  - Descriptive comment tag to expalin code blocks
  - Removed redundant tag from < h1 > tag
  - < div > tags into < section> tags for benefit lead, cost, brand 
  - Non HTML breaks for < p > tags text 
  - Inputted website title 

* CSS 
  - Chronological coding to imitate deployed webpage 
  - Consolidated class ID's with same key: values. 
  - Descriptive comments
